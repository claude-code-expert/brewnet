Setup Steps (8 Steps):
  1. Create a free Cloudflare account (dash.cloudflare.com)
  2. Visit DigitalPlat FreeDomain
  3. Register / login to DigitalPlat
  4. Search for your desired domain name
  5. Register domain (select TLD: .dpdns.org recommended)
  6. Set Cloudflare nameservers in DigitalPlat panel
  7. Wait for DNS propagation (15min ~ 24hrs, verify: dig +short NS your-domain)
  8. Return here and enter your domain name above
