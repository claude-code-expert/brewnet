Troubleshooting:
• DNS propagation can take 15 min to 48 hours after domain registration
• Check dnschecker.org for global DNS propagation status
• Ensure Cloudflare domain status shows Active
• Run docker logs brewnet-cloudflared --tail 50 to check tunnel logs
• 502 Bad Gateway: Service container not running or port mismatch — check docker ps
• SSL infinite redirect: Set Cloudflare SSL mode to Full (not "Flexible")
• Tunnel drops: Token expired or network issue — regenerate token in Cloudflare dashboard
