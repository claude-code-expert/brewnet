## 🔍 자주 묻는 질문 (FAQ)

### Q1: 정말 2주 안에 완료 가능한가?
**A**: 
- 병렬 작업 가정 시 가능
- 필수 항목 (UX-01~03)만 2주
- 모든 항목은 4주 필요
- 각 팀의 경험도에 따라 ±1주 변동

### Q2: 개선 후에도 안 되면?
**A**:
- 단계별 테스트로 위험 최소화
- 주간 리뷰로 조기 발견
- 문제 발생 시 즉시 대응
- Post-MVP에서 추가 개선 예정

### Q3: 사용자 테스트는 언제?
**A**:
- Phase 1 완료 후 (2주 후)
- 3명 이상의 초보 사용자
- 실제 환경 (자신의 시스템)
- 약 1시간 소요

### Q4: 메트릭을 어떻게 측정?
**A**:
```typescript
// 추적 코드 예시
logger.info('setup-started', { timestamp, platform });
logger.info('step-2-completed', { timestamp, service_count });
logger.info('setup-completed', { timestamp, total_time });
logger.error('setup-failed', { timestamp, step, error });

// 분석
├─ 완료율 = (setup-completed / setup-started) × 100
├─ 소요시간 = setup-completed.timestamp - setup-started.timestamp
└─ 중단율 = (setup-failed / setup-started) × 100
```

### Q5: 외부 서비스 (Cloudflare) 연동은?
**A**:
- OAuth 앱 등록 (Cloudflare 대시보드)
- API 키 발급 (DigitalPlat)
- 테스트 계정으로 먼저 검증
- 운영 계정은 정식 배포 후
